# GIT-GITHUB

Git と GitHub の使い方を学ぶための練習用リポジトリです。

## 概要

このリポジトリは、Git/GitHub の基本操作（ブランチ作成、コミット、プルリクエストなど）を実践的に学ぶために作成されました。

## 含まれるファイル

| ファイル名 | 内容 |
|---|---|
| `pokki-mas.txt` | ブランチ・PR 練習用ファイル |
| `kの子の山が好き.txt` | ブランチ・PR 練習用ファイル |
| その他 `.txt` ファイル | Git 操作の練習用ファイル |

## 学習内容

- リポジトリの作成・クローン
- ブランチの作成・切り替え
- ファイルの追加・コミット
- プルリクエストの作成・マージ
- コンフリクトの解消

## 使い方

```bash
# リポジトリをクローン
git clone https://github.com/ToshihiroMaebashi/GIT-GITHUB.git

# ブランチを作成して切り替え
git checkout -b 新しいブランチ名

# 変更をコミット
git add .
git commit -m "コミットメッセージ"

# リモートにプッシュ
git push origin ブランチ名
```
